# Kobiyim
Kobiyim, Kobi'lerin veri akışını tek yerde toplayarak bu bilgilerin gerekli gözüken noktalarda, gerekli kişilerle paylaşımlarını yaparak dijitalleşme süreçlerinde ilk adım olmayı hedeflemektedir.
## İçindekiler
## Dahil Edilecekler
* Kullanıcı işlemleri
* Hareket kayıtları
* Otomatik yedekleme
## Kurulum
Paketin kurulumu için Laravel kullanmanız gerekmektedir. Laravel kurulu dizine ```composer require kobiyim/kobiyim``` komutunu çalıştırarak paketi sisteminize dahil edebilirsiniz.
